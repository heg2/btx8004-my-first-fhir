# btx8004-my-first-fhir
This is a template for a very basic JS webapp for using FHIR.

## Prerequisites:
You need to have [Node.js](https://nodejs.org/en/download) (required) and [Git](https://git-scm.com/install/) (recommended) installed on your computer.

## Set up:
Clone this repository by entering the following command to your terminal:
```bash
git clone https://github.com/heg2/btx8004-my-first-fhir
```
If you don't want to use Git to clone the repository, you can also download the code [as a ZIP file](https://github.com/heg2/btx8004-my-first-fhir/archive/refs/heads/main.zip).

When you successfully cloned the repository, change into the working directory using:
```bash
cd btx8004-my-first-fhir
```

Now you have to install the dependencies with:
```bash
npm install
```

When all these steps where successful, you can now start the app by entering:
```bash
npm run start
```

This starts a local development server that hosts your web app. Open [localhost:5173](http://localhost:5173/) in your browser to see the app. The development server is live reloading, so if you save any changes in the source files, the app in the browser will automatically update.

To stop the server, you can press `[ctrl] + [c]` in the terminal window; or just close the terminal window.
